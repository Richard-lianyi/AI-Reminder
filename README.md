# AI Reminder Assistant

A beginner-friendly AI reminder assistant built with FastAPI and SQLite.  
This project allows users to create, store, and manage reminders, and serves as a foundation for future AI-based natural language scheduling.

## Features
- Add reminders via API
- Store reminders using SQLite database
- Backend built with FastAPI
- Modular structure (database, API, checker)

## Tech Stack
- Python
- FastAPI
- SQLite

## Future Improvements
- Natural language input (e.g., "Remind me tomorrow at 3 PM")
- Notification system
- Frontend UI integration
